Interface Extends

Extends is a type-level function that takes in two types, T and U, and returns a boolean that represents whether T extends U.

Template: T

The supertype that we are checking if U extends.

Template: U

The type that we are checking if it is a subtype of T.

Example

For example, we can use Extends to determine whether a type T extends a type U. In this example, we test whether true extends boolean:

We apply Extends to boolean and true respectively using the $ type-level applicator. This checks whether true extends boolean:

import { $, Conditional } from "hkt-toolbelt";

type Result = $<$<Conditional.Extends, boolean>, true>; // true

In the following examples, we test whether a string extends a number:

import { $, Conditional } from "hkt-toolbelt";

type Result = $<$<Conditional.Extends, number>, string>; // false

Note that TypeScript provides its own extends keyword that can be used to perform a subtype check. However, extends can only be used in a conditional type. Extends is composable, so it can be used in more sophisticated type-level functions.

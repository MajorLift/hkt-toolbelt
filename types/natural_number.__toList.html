A type-level utility that recursively breaks down a number into a list of its digits. This utility is primarily used within the NaturalNumber.ToList interface.

Type Parameters

  • S extends Number

    The number to be converted into a list of digits.

  • O extends string[] = []

    An accumulator used during the recursive breakdown process. Defaults to an empty list.

Returns

A list of digits representing the number. If the number is zero, it returns a list with a single '0'.

Example

// Convert the number 42 into a list of digits:
type ListOfDigits = _$toList<42>; // This will be inferred as ['4', '2']
